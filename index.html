  1. Get your Discord Data Package. Here's how to do it.

What is it? It is a site (program) that graphs your Discord data from a file from your Discord Data Package, fully on-device.
Why is it? I wanted a write a simple site to point my friends to to view their Discord data. Cause it is fun to look at what Discord thinks about your gender and age.
Not everyone has python installed, and this site is basically equivalent to a python script, but it runs in a browser, and works fully locally.
How does it work? You provide a directory, a Python script opens the events-*.json file and plots the data with matplotlib. And it doesn't send nor store any of your data! Thanks to rebane2001 for writing a gist by which this plotting script is heavily inspired.
Under the hood it uses PyOdide runtime (WASM) which conveniently bundles several popular Python packages, including matplotlib.
Huh? So it works fully on-device? You can see the source code for yourself. It pulls PyOdide runtime from a CDN, and the python script is in /script.py, so feel welcome to check it out yourself.
